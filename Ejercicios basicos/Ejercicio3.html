Que es una página web?

Es una colección de textos y elementos multimedia (como por ejemplo imágenes)

En una página el PageRankTM (PR) es un valor numérico que representa la importancia que una página web tiene para Google.

Las paginas web de forma general se pueden clasificar en dos tipos: Páginas web Estáticas y Página Web Dinámicas
